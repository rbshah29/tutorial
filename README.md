
# user-details

## docker variables

| Variable | Usage |
| -------- | ----- |
| MONGO_URL | -e MONGO_URL=<<_your_mongo_url_>> |
| LOG_LEVEL | -e LOG_LEVEL=<<_your_log_level_>> |
| PORT | -e PORT=<<_your_port_>> |
| DATABASE | -e DATABASE=<<_your_database_>> |
| JSON_LIMIT | -e JSON_LIMIT=<<_your_json_limit_>> |
| LOGS | -v <<_your_path_>>:/logs |
